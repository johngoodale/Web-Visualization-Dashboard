The purpose of this project was to analyze how weather changes as you approach the equator. To answer that question I created a Python script to visualize the weather of 500+ cities across the world of varying distance from the equator. The data was sourced from the the OpenWeatherMap API.

After compiling the dataset, visuals were created using the Python library Matplotlib. This allowed me to visually display the data with various weather characteristics vs. Latitude (Max Temperature, Humidity, Cloudiness, and Wind Speed). This web dashboard provides the source data and visualizations that were created in analyzing the random city weather data and descriptions of any trends and correlations encountered.
